# 说明

DocumentDirectoryParser类是一个用于解析目录文件的工具，具备多种功能。它支持模式匹配，可以根据特定模式筛选文件。此外，该类能够加载隐藏文件，确保不遗漏任何重要数据。递归遍历功能使其能够深入子目录，全面解析所有文件。文件抽样功能则允许用户从大量文件中抽取部分样本进行分析。这些特性使得DocumentDirectoryParser类在处理复杂目录结构时非常高效和灵活。
