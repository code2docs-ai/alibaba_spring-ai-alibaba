# 说明

Swagger配置类用于管理API分组，并指定该分组匹配路径为/graph-studio。该配置类的主要功能是通过Swagger工具对API进行分组管理，确保与/graph-studio路径相关的API能够被正确分类和展示。通过这种方式，开发者可以更方便地管理和查看与该路径相关的API文档，提升API的可维护性和可读性。
